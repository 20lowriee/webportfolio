Web Accessibility Checklist

User Characteristics Accessible Design Tip
1. Unable to See 1. Code images with ALT text
2. Unable to Percieve Colors 2. Avoid using color alone to convey information
3. Unable to Use Mouse 3. Be sure all features can be accessible via keyboard
4. Unable to Hear 4. Add captions to multimedia
5. Seizure Prone 5. Avoid flashing content
6. Easily Distracted 6. Keep the design simple
