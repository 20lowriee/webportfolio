My Web Authoring Software

The web authoring software I am using is Adobe Dreamweaver. I really like this program because it is easy to use and there are a lot of features available. For example, you can view and edit the web page within the actual code or on the live view. Also, the software shows the lines in the code that has an error so that it can be easily fixed. In addition to all of this, it is easy to edit and add elements to the web page. Overall, I really like this program and hope to use it for more projects in the future.
