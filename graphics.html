Evaluation of Web Graphics

Website I - Apple, Inc.

  • The purpose of the graphics on this website are to show the company’s products and make them appealing for potential buyers.
  • The graphics on this website enhance the user experience and have special features. For example, if a user scrolls down on a
    page of the website, a graphic will move or become a different photo. I think this is a cool way to show products and help
    increase online sales of products.
  • The graphics enhance the design of the website and meet many of the guidelines in the article. For example, the graphics fit in
    with the site’s purpose, which is to sell the products and show all tech specs, the website loads quickly, the graphics are not too
    distracting and the focus is still on the main content, the photos are not too obnoxious and there is a website theme, all content
    can still be viewable and understandable without the graphics, and most text on the site is black on white, which provides great
    contrast for the viewer. This website meets all of the website graphic guidelines.

Graphic File Types

Image I

For this image of a graph, I believe it would be the most beneficial to save this image as an SVG file. This is because this image may need to
be displayed on many platforms, in various sizes. This information will need to be delivered to anyone because it is statistical data dislayed
on a graph.

Image II

For this next image of a monkey, I think it should be saved as a JPG file. JPG files are used mostly for photography with colors and this
photo meets both of those standards.

Image III

For this image of a mountain, I think it should be saved as a JPG file. I think this because it is photography that uses many colors, and JPG
files are great for these types of images.

Image IV

For this next image of puzzle pieces, I think it should be saved as a PNG-8 file. This is because these files are good for icons and transparent
images without many colors, and these images of puzzle pieces are solid colors and are also web icons.

Image V

For the last image of a stop sign, I think it should be saved as a PNG-8 file. PNG-8 files are good for photos with a transparent background,
which this image does have.
